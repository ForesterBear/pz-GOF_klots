# Command Pattern (Приклад)

Коротко: Command інкапсулює запит як об'єкт, дозволяючи параметризувати клієнтів різними запитами, ставити запити в чергу або реєструвати їх.

Файли:
- `command.ts` — інтерфейс `Command` і конкретні команди (`TurnOnCommand`, `TurnOffCommand`)
- `receiver.ts` — `Receiver` з операціями, які виконуються командами
- `invoker.ts` — `Invoker`, який зберігає і виконує команду
- `index.ts` — функція `runCommand()` для демонстрації

Приклад запуску: `npm start` → у виводі побачите послідовність включення та вимкнення через виклики команд.
