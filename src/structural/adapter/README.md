# Adapter Pattern (Приклад)

Коротко: Adapter дозволяє використовувати інтерфейс класу (Adaptee) у вигляді, очікуваному клієнтом (Target), адаптуючи виклики.

Файли:
- `target.ts` — інтерфейс, який очікує клієнт
- `adaptee.ts` — клас з несумісним інтерфейсом
- `adapter.ts` — адаптер, що перетворює виклики
- `index.ts` — функція `runAdapter()` для демонстрації

Приклад запуску: `npm start` → вивід показує вихід `adaptee` і адаптований результат.
