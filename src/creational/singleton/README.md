# Singleton Pattern (Приклад)

Коротко: Singleton гарантує, що клас має лише один екземпляр, і надає глобальну точку доступу до нього.

Файли:
- `singleton.ts` — реалізація Singleton
- `index.ts` — функція `runSingleton()` для демонстрації

Приклад запуску: `npm start` → у виводі побачите демонстрацію, що обидві змінні посилаються на один і той самий екземпляр.
