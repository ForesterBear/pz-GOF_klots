# Prototype Pattern (Приклад)

Коротко: Prototype дозволяє створювати нові об'єкти шляхом копіювання існуючих, замість створення з нуля.

Файли:
- `prototype.ts` — інтерфейс `Prototype<T>` та клас `Product` з реалізацією `clone()`
- `index.ts` — функція `runPrototype()` для демонстрації

Приклад запуску: `npm start` → у виводі відобразиться оригінал та копія з модифікаціями, що підкреслює незалежність екземплярів.
